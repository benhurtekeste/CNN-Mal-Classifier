# This project is part of the AI Red Teamer path on Hack The Box
### Based on their characteristics, modus operandi and propagation methods, malware belong to a certain family
### This project aims to advanced alternative to manual malware analysis which includes the arduous process of reverse engineering
### The ResNet50 model classifies a malware based on its image-representation derives from the malware binary 
### Malimg Original Dataset is used for training
### This project deals with malware targetting Windows OS
