# This project is part of the AI Red Teamer path on Hack The Box
### Based on their characteristics, modus operandi, and propagation methods, a given malware belongs to a particular family
### This project aims to be an advanced alternative to manual malware analysis, which includes the arduous process of reverse engineering
### The ResNet50 model classifies malware based on its image representation derived from the malware binary 
### Malimg Original Dataset is used for training
### This project deals with malware targetting Windows OS
### Note: Remaining tasks - since the original dataset seems unbalanced, I will look for ways to balance the training data.
